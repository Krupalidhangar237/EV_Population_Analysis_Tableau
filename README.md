This project presents an end-to-end **Visual Analytics solution** to analyze the growth, distribution, and adoption trends of **Electric Vehicles (EVs)** across the United States. ## üßπ Data Cleaning & Preparation
Data preprocessing was performed using **Excel** and **Tableau** before visualization:

- Removed duplicates and handled missing values
- Standardized state and city names for accurate mapping
- Created derived fields:
  - Vehicle Category (BEV / PHEV)
  - Decade (from Model Year)
  - EV Age (Current Year ‚Äì Model Year)
- Filtered only analytics-relevant columns
